# Castle of Advantis

A fast-paced 2D combat-platformer built in GameMaker Studio 2. Choose your class, fight through rooms of enemies, and chase the highest score possible. 

> **Project Status:** This game is currently in an incomplete demo/prototype state. Features, balancing, and assets are subject to change.
> Play WIP Demo:  https://gx.games/games/mhvax2/castle-of-advantis/tracks/cd4cbaf6-0c71-45ce-a210-17c39f1a922b/

---

## Features

* **5 Playable Classes:** Distinct playstyles with unique basic attacks and special abilities.
* **Room-to-Room Progression:** Clear out waves of enemies to advance to the next challenge.
* **Score System:** Defeat enemies and chain together clean combat rooms to rack up points.
* **Classic 2D Platforming:** Responsive movement, jumping, and combat mechanics.

---

## The Classes

| Class | Weapon/Style | Attack / Ability | Playstyle |
| :--- | :--- | :--- | :--- |
| **Knight** | Greatsword | Medium Slash / Sword Throw | Balanced, high survivability |
| **Rogue** | Dual Daggers | Fast Slice / Incorporeal Form | High speed, high risk/reward |
| **Shieldbearer** | Shield and Spear | Spear Poke / Reflective Shield | Slow, defensive powerhouse |
| **Archer** | Bow & Arrow | Piercing Shot / Ensnaring Arrow | Long-range kiting & precision |
| **Mage** | Magic Staff | Magic Orb / Fireball | Glass cannon, massive ranged damage |

---

## Controls (Default)

* **Move Left / Right:** `A` / `D`
* **Jump:** `Spacebar`
* **Basic Attack:** `Left Click`
* **Special Ability:** `Right Click`
* **Menu Navigation / Select:** `A` / `D` / `Spacebar`

---

## Built With

* **Engine:** GameMaker Studio 2 (GMS2)
* **Language:** GML (GameMaker Language)
